# Metrics

Measurements that help teams discuss architecture with evidence.

Metric entries define measurements used in architecture review. IDs use the `METRIC-###` prefix and are assigned in `../index.yaml`.

Each concept file must include YAML front matter with `id`, `type`, `status`, and `canonical_name`.

Relationships are maintained in `../index.yaml`.
