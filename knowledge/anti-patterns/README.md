# Anti-patterns

Recurring structures that create long-term engineering cost.

Anti-pattern entries define recurring harmful structures and recovery guidance. IDs use the `ANTIPATTERN-###` prefix and are assigned in `../index.yaml`.

Each concept file must include YAML front matter with `id`, `type`, `status`, and `canonical_name`.

Relationships are maintained in `../index.yaml`.
