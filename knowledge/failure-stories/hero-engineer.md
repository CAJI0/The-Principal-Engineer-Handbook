# Hero Engineer

## Definition

A system depends on one engineer's private memory and intervention.

## Why It Matters

Heroics can hide missing architecture, ownership, and documentation.

## Symptoms

- Incidents wait for one person.
- Reviews defer to memory instead of artifacts.
- The system is stable only while the expert is available.

## Example

Only one engineer knows the safe order for factory recovery after a failed update.
