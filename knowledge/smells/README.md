# Smells

Early signs that architecture is becoming harder to change.

Smell entries define warning signs and recovery guidance. IDs use the `SMELL-###` prefix and are assigned in `../index.yaml`.

Each concept file must include YAML front matter with `id`, `type`, `status`, and `canonical_name`.

Relationships are maintained in `../index.yaml`.
