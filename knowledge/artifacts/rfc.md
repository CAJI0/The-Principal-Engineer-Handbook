# RFC

## Purpose

Invite review before a decision becomes hard to change.

## When To Use

Use an RFC for proposals that need feedback across ownership boundaries.

## Template

- Summary
- Motivation
- Scope
- Non-goals
- Proposal
- Risks
- Open questions

## Common Mistakes

- Treating the RFC as approval theater.
- Omitting non-goals.
- Asking for review after implementation is effectively complete.
