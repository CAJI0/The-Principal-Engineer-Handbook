# Artifacts

Reusable engineering documents and records.

Artifact entries define documents, ledgers, catalogs, or templates used by engineering work. IDs use the `ARTIFACT-###` prefix and are assigned in `../index.yaml`.

Each concept file must include YAML front matter with `id`, `type`, `status`, and `canonical_name`.

Relationships are maintained in `../index.yaml`.
