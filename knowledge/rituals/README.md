# Rituals

Recurring practices that keep architecture visible.

Ritual entries define repeated engineering practices. IDs use the `RITUAL-###` prefix and are assigned in `../index.yaml`.

Each concept file must include YAML front matter with `id`, `type`, `status`, and `canonical_name`.

Relationships are maintained in `../index.yaml`.
