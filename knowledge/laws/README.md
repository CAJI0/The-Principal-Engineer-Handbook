# Laws

Core engineering laws used by the handbook.

Law entries define durable engineering rules. IDs use the `LAW-###` prefix and are assigned in `../index.yaml`.

Each concept file must include YAML front matter with `id`, `type`, `status`, and `canonical_name`.

Relationships are maintained in `../index.yaml`.
