# Vocabulary

Reusable terms used throughout the handbook.

Vocabulary entries define shared language. IDs use the `VOCAB-###` prefix and are assigned in `../index.yaml`.

Each concept file must include YAML front matter with `id`, `type`, `status`, and `canonical_name`.

Relationships are maintained in `../index.yaml`.
