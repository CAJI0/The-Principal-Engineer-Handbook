# Code of Conduct

This project uses a simple contributor standard until the authors select a formal code of conduct.

## Expected Behavior

- Be respectful and constructive.
- Critique work, not people.
- Preserve author intent.
- Make review comments specific and actionable.
- Assume editorial care is part of the work.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private author or reviewer information.
- Adding content that misrepresents an author's position.
- Ignoring explicit author review boundaries.

## Reporting

Report conduct concerns to the repository maintainers. Maintainers should handle reports privately, promptly, and with care.
