# Canon

The canon records recurring concepts approved for the book project.

## Knowledge Model

The handbook is supported by PEAK: Principal Engineering Architecture Knowledge.

PEAK organizes the handbook's concepts into vocabulary, laws, artifacts, rituals, smells, anti-patterns, metrics, and failure stories.

The canonical knowledge base lives in [`knowledge/`](../knowledge/README.md).

## Stable Knowledge IDs

All canonical concepts should receive stable IDs.

Examples:

- `LAW-001`
- `SMELL-001`
- `ARTIFACT-001`
- `RITUAL-001`

IDs are never reused.
