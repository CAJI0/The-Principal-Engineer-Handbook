# ADR-002: Use Handbook Format

## Status

Accepted

## Context

The project needs a clear book form before full chapter writing begins.

## Decision

This project is a handbook/playbook, not a linear textbook.

## Consequences

- Readers can use sections as practical references.
- Chapters can include artifacts, checklists, reviews, and decision records.
- Maintainers must preserve handbook usability when organizing material.

## Alternatives Considered

- Write as a linear textbook.
- Write as a collection of independent essays.
